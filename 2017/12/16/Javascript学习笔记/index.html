1.3数据类型转换

  • 字符串到数值。再从字符串转到数值时，+运算和-运算是不同的。对于+运算，如果操作数中含有字符串值，她就将它变成字符串连接运算。
  • 数值到字符串。一般分为显式类型转换和隐式类型转换，String函数和toString()方法称为显式类型转换；隐式类型转换通常使用+运算符。 
          //惯用的数据类型转换方法
      function(){
          //从数值转为字符串
          var n = 3;
          n + '';      //将数值3转换为字符串值'3'
          //从字符串到数值
          var s = 3;
          +s;          //将字符串值'3'转换为数值3
      }
  • 布尔类型转换。在进行布尔型的数据类型转换时，应当对Object类型的情况多加注意。Object在被转换为布尔型之后必定为true。以下代码清单1的结果都是显示T。而像代码清单2这样，通过函数调用方式获取的结果就不再是Object类型，而是相应的内建类型了。
    //代码清单-1
var b = new Boolean(false);    //Object类型
var z = new Number(0);         //Object类型
var s = new String('');        //Object类型
!!b;!!z;!!s;                   //true

      

    //代码清单-2
var b = Boolean(false);  //Boolean类型
var z = Number(0);         //Number类型
var s = String('');         //String类型
!!b;!!z;!!s;             //false

    2 语句、表达式和运算符

    2.1 语句

  • with语句。with语句用于临时改变名称（变量名或函数名）的查找范围。with语句中使用的是Object类型的。如果使用其他类型则会被转换为Object类型。在with语句中对变量名进行查找时，将会从所指定对象的属性开始查找。 
          //with使用示例
      var x = 1;              //全局变量
      var obj = {x: 7; y: 8}; //Number类型
      with(obj){
          print(x);          //如果要查找x，则会在查找全局变量x之前先查找到obj.x
      }
